# Late Delivery Risk Prediction

This project transforms APL Logistics’ supply chain operations from reactive delay management to predictive risk intelligence. Unlike earlier projects, which analyze past performance and financial outcomes, this project anticipates future delivery failures, enabling early action and smarter logistics decisions. It forms the predictive backbone of an intelligent, data-driven supply chain.

## Setup

1. Install dependencies:
   ```bash
   pip install pandas scikit-learn joblib xgboost
   ```

2. Train the model:
   ```bash
   python train_late_delivery_model.py
   ```

3. Make predictions:
   ```bash
   python predict_late_delivery_risk.py APL_Logistics.csv
   ```

4. Compare both models and save the comparison output:
   ```bash
   python compare_models.py
   ```

## Files

- `APL_Logistics.csv`: Dataset
- `train_late_delivery_model.py`: Training script
- `predict_late_delivery_risk.py`: Prediction script
- `compare_models.py`: Random Forest vs. XGBoost comparison script
- `late_delivery_risk_model_xgb.joblib`: Small saved XGBoost pipeline included for prediction
- `late_delivery_risk_model.joblib` and `late_delivery_risk_model_rf.joblib`: excluded from the repo due to size; generate locally by running `python train_late_delivery_model.py`
- `predictions.txt`: Latest prediction output file
- `compare_results.txt`: Latest comparison output file

## Example terminal commands

Run prediction and save the output to `predictions.txt`:
```bash
python predict_late_delivery_risk.py APL_Logistics.csv predictions.txt
```

Run the model comparison and save the output to `compare_results.txt`:
```bash
python compare_models.py
```

Then open `predictions.txt` or `compare_results.txt` to see the results.
